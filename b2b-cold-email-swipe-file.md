## HOW TO USE THIS SWIPE FILE

## BONUS: THE ANATOMY OF A HIGH-CONVERTING COLD EMAIL

| Element | What to Do |
|--------|------------|
| Subject line | Short, personal, curiosity-driven. Under 7 words. |
| Opening line | Reference something specific to THEM. Not your company. |
| Pain point | Name a real problem they likely have. |
| Proof | Drop a result. Customer name + metric. |
| CTA | One clear ask. A question or a time. Never multiple CTAs. |
| Length | Under 100 words for first touch. |
